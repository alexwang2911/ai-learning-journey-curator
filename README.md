# AI Learning Journey Curator 📚🤖

Proyek ini adalah manifestasi dari filosofi **"Learn AI Together"**. Alat ini dirancang untuk membantu para profesional membangun kurikulum belajar mandiri yang terstruktur di bidang AI tanpa terjebak dalam jargon teknis yang membingungkan.

## 🌟 Fitur Utama
- **Buzzword-Free Mapping:** Menyederhanakan konsep AI yang kompleks menjadi jalur pembelajaran yang mudah dipahami.
- **Workforce Alignment:** Menyesuaikan materi belajar dengan kebutuhan evolusi tenaga kerja saat ini.
- **Open Access Resources:** Mengintegrasikan ribuan sumber daya gratis dan alat produktivitas yang sudah dikurasi.
- **Community-Driven:** Dirancang dengan prinsip inklusivitas seperti yang diperjuangkan di AI4D.

## 💻 Tech Stack
- **Backend:** Python (FastAPI)
- **Data Handling:** Pandas (untuk kurasi dataset edukasi)
- **Frontend:** Streamlit (untuk aksesibilitas cepat)
- **AI:** GPT-based categorization untuk menyaring 'buzzwords'.
